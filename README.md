# Public GenAI Deployment Templates

Dette repository indeholder skabeloner til udrulning af en offentlig generativ AI løsning på en [Kubernetes](https://kubernetes.io/da/docs/concepts/overview/what-is-kubernetes/) cluster.

### Indhold

``` 
public-genai-deployment-templates/

  ├── /charts - [Helm](https://helm.sh/docs/intro/)-charts til udrulning på k8s

  └── /docs - Dokumentation for brug og deling af skabelonerne
```
